## Description
React Component Library for visualizing large amounts of things.

## Local Development 
### Dependencies 
To install dependencies with npm, run:
```
$ npm install
```

### Linting
To run the linter once:
```
npm run lint
```

To run the linter watch task:
```
npm run lint:watch
```
